# Installation
1. Installation:

    git clone git@github.com:CivilizationV/dotvim.git ~/.vim

2. Create symlinks:

    ln -s ~/.vim/vimrc ~/.vimrc

3. Switch to the `~/.vim` directory, and fetch submodules:

    cd ~/.vim
    git submodule init
    git submodule update
